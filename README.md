# KeenKeeper

# 📌 Description
KeenKeeper is a personal relationship management app that helps people stay in touch with the people that matter most. It allows to monitor interactions, set communication goals, and maintain meaningful relationships.

# 🛠️ Built With
- Html
- CSS (Tailwind CSS)
- JavaScript (React, Next.js)
- Chart.js (for analytics visualization)

# ✨ Features
1. **Handling Friends**  
   Build and organize contacts with status labels like "On Track" or "Needs Attention".

2. **Tracking Interactions**  
   Track your communication history by logging calls, texts, meetups, and viewing them in a timeline.

3. **Analytics & Reminders**  
   Visual analytics deliver insights to help you set connection goals, and improve how often and how you connect.
